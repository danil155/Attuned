
### Идентификация
| Поле        | Тип данных | Описание                                                     |
|-------------|------------|--------------------------------------------------------------|
| `id`        | Integer    |                                                              |
| `name`      | String     | Полное официальное название игры                             |
| `slug`      | String     | Подходящая для url-адресов версия названия в нижнем регистре |
| `summary`   | String     | Краткое описание игры                                        |
| `cover.url` | String     | Ссылка на обложку игры                                       |
| `storyline` | String     | Описание сюжета игры                                         |
| `url`       | String     | Ссылка на страницу игры на igdb.com                          |

---

### Классификация
| Поле          | Тип данных    | Описание                                                                  |
|---------------|---------------|---------------------------------------------------------------------------|
| `game_type`   | Enum(Integer) | Цифровые категории к которой относится игра (основная игра, DLC и прочее) |
| `game_status` | Enum(Integer) | Статус релиза (вышла, альфа, снята и т.д.)                                |

---

### Ключевые данные для рекомендаций
| Поле                       | Тип данных | Описание                                                           |
|----------------------------|------------|--------------------------------------------------------------------|
| `genres.name`              | List       | Жанры, к которым относится игра                                    |
| `themes.name`              | List       | Сеттинг игры - более широкие категории, чем жанры                  |
| `keywords.name`            | List       | Ключевые слова                                                     |
| `game_modes.name`          | List       | Режимы игры (соло, мультиплеер и т.д.)                             |
| `played_perspectives.name` | List       | От какого лица вид (первое, третье лицо, разделенный экран и т.д.) |
| `platforms.name`           | List       | Платформы поддерживающие игру (PC, PS и т.д.)                      |
| `involved_companies`       | List       | Компании-разработчики (берем для связи игр по студии разработки)   |

---

### Метрики популярности
| Поле                      | Тип данных | Описание                                            |
|---------------------------|------------|-----------------------------------------------------|
| `rating`                  | Float      | Средний рейтинг игроков (0 - 100)                   |
| `rating_count`            | Integer    | Общее количество оценок                             |
| `aggregated_rating`       | Float      | Средний рейтинг внешних изданий-критиков (0 - 100)  |
| `aggregated_rating_count` | Integer    | Общее количество оценок изданий-критиков            |
| `hypes`                   | Integer    | Количество юзеров, добавивших игру в hype до релиза |

---

### Даты
| Поле                       | Тип данных | Описание                                       |
|----------------------------|------------|------------------------------------------------|
| `first_release_date`       | Integer    | Первая дата выхода игры                        |
| `updated_at`               | Integer    | Последняя дата обновления записи о игре в IGDB |
